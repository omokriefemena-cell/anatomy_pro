# Anatomy Pro - Official Anatomy Content Synopsis

## Purpose

This document defines the official structure for every anatomical topic in Anatomy Pro.

Every lesson in the application should follow this structure whenever applicable to ensure consistency throughout the platform.

Not every topic will require every section. However, the order should remain consistent.
