# Anatomy Pro

## Vision

Anatomy Pro is a premium offline-first anatomy learning platform built for medical, anatomy, nursing, physiotherapy and allied health students.

The goal is to become one of the world's most comprehensive anatomy learning applications.

---

# Mission

Provide accurate, detailed, visually appealing and clinically relevant anatomy education that is accessible to every student.

# Core Principles

Every feature must be

- reusable
- scalable
- maintainable
- fast
- beautiful
- educational

Never build shortcuts that will create technical debt later.

---

# Coding Principles

Avoid duplicate code.

Prefer reusable widgets.

Keep business logic outside UI.

Use models and services.

Follow clean architecture whenever practical.

Every screen must be data-driven.

# Anatomy Content Standard

Every anatomical structure should follow a standard synopsis.

Whenever applicable include:

1. Introduction
2. Definition
3. Overview
4. Classification
5. Location
6. Surface Anatomy
7. Anatomical Features
8. Relations
9. Blood Supply
10. Venous Drainage
11. Lymphatic Drainage
12. Nerve Supply
13. Histology
14. Embryology
15. Ossification
16. Functions
17. Attachments
18. Applied Anatomy
19. Clinical Anatomy
20. Anatomical Variations
21. Imaging
22. Mnemonics
23. High Yield Facts
24. Examination Questions
25. References
